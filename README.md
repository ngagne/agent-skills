# agent-skills

A growing collection of **AI agent skills** I’ve built.

I use this repo to keep skill folders, helper assets, and supporting tooling for my own agentic workflows. Each skill is meant to be self-contained, easy to inspect, and practical to reuse.

## What's here today

| Path | Purpose |
| --- | --- |
| `skill-creator/` | A skill for creating, revising, validating, and evaluating other skills |

## Repository shape

Skills in this repo live in their own top-level directories and usually follow a structure like this:

```text
skill-name/
├── SKILL.md
├── scripts/
├── references/
├── assets/
└── ...
```
